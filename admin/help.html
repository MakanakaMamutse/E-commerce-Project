Order Management Guide

Understanding Order Status
not paid Payment pending
paid Payment confirmed
delivered Order completed
How to Update Orders
  1. Navigate to the Orders page from the sidebar
  2. Find the order you want to update
  3. Click the blue "Edit" button in the actions column
  4. Update the order status or other details
  5. Save your changes

Product Management Guide

Adding New Products
  1. Click "Add New Product" from the sidebar
  2. Fill in all required product information:
    • Product name and description
    • Price and category
    • Stock quantity
    • Product images
  3. Set the product status (active/inactive)
  4. Save the product
Managing Existing Products
  • Use the Products page to view all items
  • Edit product details by clicking the edit button
  • Update stock levels regularly
  • Deactivate products that are out of stock

Frequently Asked Questions

To process a refund:

  1. Locate the order in the Orders section
  2. Verify the refund request is valid
  3. Contact your payment processor to initiate the refund
  4. Update the order status to "refunded"
  5. Notify the customer about the refund processing

To update inventory:

  1. Go to the Products page
  2. Find the product you want to update
  3. Click the edit button
  4. Update the stock quantity field
  5. Save the changes

The system will automatically show "Out of Stock" when quantity reaches zero.

If you're having trouble accessing your account:

  • Check your username and password carefully
  • Use the password reset feature if available
  • Clear your browser cache and cookies
  • Try using a different browser
  • Contact technical support if the problem persists

To export order data:

  1. Go to the Orders page
  2. Use the Filter button to select specific orders if needed
  3. Click the Export button
  4. Choose your preferred format (CSV, Excel)
  5. The file will be downloaded to your computer
